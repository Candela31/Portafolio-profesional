# Proyecto integrador: Portafolio profesional y academico



# Descripcion
Este proyecto consiste en una página web  diseñada para exhibir mis proyectos profesionales y académicos. El objetivo principal es presentar mi perfil brindando un acceso rápido a mis trabajos de bases de datos, lógica de programación y desarrollo web.

# Mejoras visuales
- Agregamos el CSS.
- variables, paletas de colores, fuentes, fondos
- Layout con Grid en la seccion principal
- Lista de navegacion con flexbox
- efecto hoover con tansicion-
- Card von border con border-radius, box-shadow  
- Botones estilizados con transición de color 
- Animación de rebote en la foto de perfil con @keyframes
- Diseño responsive con media queries: tablet (768px) y celular (480px)
- Menú que se reorganiza verticalmente en pantallas pequeñas

# Funcionalidades implementadas con JavaScript
- Archivo `script.js` externo, vinculado desde `index.html` y `proyecto.html`.
- Renderizado dinámico de la lista de habilidades a partir de un array.
- Renderizado dinámico de las tarjetas de proyectos a partir de un array de objetos.
- Buscador de proyectos en tiempo real (evento `keyup`) que filtra el array de proyectos por título o descripción.
- Formulario de contacto con validación (campos obligatorios, formato de email y longitud mínima del mensaje) usando `try/catch` para el manejo de errores.
- Efecto interactivo sobre la foto de perfil con los eventos `mouseover` / `mouseout`.
- Manipulación del DOM: creación de elementos, modificación de texto, agregado de clases y contenido condicional ("no se encontraron proyectos").

# Conclusion

-  Implementar conocimientos dados en clase.
-  Organizar información.
-  Incorporar interactividad real mediante JavaScript y manipulación del DOM.
