# Proyecto integrador: Portafolio profesional y academico



# Descripcion
Este proyecto consiste en una página web  diseñada para exhibir mis proyectos profesionales y académicos. El objetivo principal es presentar mi perfil brindando un acceso rápido a mis trabajos de bases de datos, lógica de programación y desarrollo web.

# Mejoras visuales
- Agregamos el CSS.
- variables, paletas de colores, fuentes, fondos
- Layout con Grid en la seccion principal
- Lista de navegacion con flexbox
- efecto hoover con tansicion-
- Card von border con border-radius, box-shadow  
- Botones estilizados con transición de color 
- Animación de rebote en la foto de perfil con @keyframes
- Diseño responsive con media queries: tablet (768px) y celular (480px)
- Menú que se reorganiza verticalmente en pantallas pequeñas
